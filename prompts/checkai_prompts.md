# CheckAI prompts

Промпты ниже рассчитаны на веб-интерфейсы китайских моделей. Управляющая часть написана по-китайски, а итоговый документ должен быть строго на русском.

Плейсхолдеры:

- `{TRACK}` - `generation` или `rewriting`.
- `{DOMAIN}` - область из списка тем, например `Аддитивные технологии`.
- `{TOPIC}` - полная тема с сайта.
- `{SECTION}` - секция исходного фрагмента для rewriting, если она известна.
- `{FRAGMENT}` - исходный фрагмент для rewriting.
- `{SCORE_FEEDBACK}` - детали неудачной отправки, если есть.

## 1. DeepSeek Expert: генерация с нуля

```text
你现在不是聊天助手，而是俄语学术短文生成器。任务属于公开 CheckAI 比赛：文本必须完全由 AI 生成，并且必须像正常俄罗斯本科二年级学生写的科学文本，而不是像模板化的 AI 文章。

输出规则：
1. 只输出俄语正文，不要中文解释，不要前言，不要说“конечно”。
2. 主题领域：{DOMAIN}
3. 完整题目：{TOPIC}
4. 文章长度：4500-6500 русских символов с пробелами.
5. 必须使用 ровно такие основные заголовки:
Введение
Теоретическая часть / Методы
Практическая часть / Эксперименты, результаты
Заключение
6. Не добавляй список литературы, аннотацию, ключевые слова и нумерованные списки.
7. Не используй markdown-жирность, таблицы, маркированные списки.

风格要求：
- Научный стиль, но без чрезмерной гладкости.
- Пиши как студент, который понимает тему, но не пытается выглядеть профессором.
- Абзацы должны быть разной длины: где-то 2 предложения, где-то 4-5, иногда один короткий абзац.
- Предложения разной длины. Не строй весь текст одинаковым ритмом.
- Используй конкретные предметные детали, механизмы, параметры, причины, ограничения.
- Не делай широкое вступление про мир, эпоху, цифровизацию и важность темы.
- В первом абзаце сразу войди в конкретную проблему темы.
- В практической части опиши учебный или модельный эксперимент так, будто студент разбирал пример по открытым данным/типовой ситуации. Не выдумывай точные ссылки на статьи.
- В заключении дай спокойный вывод без торжественной концовки.

Запрещенные выражения в русском тексте:
«В современном мире», «В эпоху цифровых технологий», «Сегодня как никогда важно», «В условиях стремительно меняющегося мира», «Исследования показывают», «Эксперты отмечают», «По мнению специалистов», «Согласно последним данным», «Нельзя не отметить», «Стоит также упомянуть», «Важно понимать, что», «Следует отметить», «Не менее важным является», «Таким образом, можно сделать вывод», «Подводя итог», «уникальный подход», «инновационное решение», «эффективный инструмент», «комплексная стратегия», «качественный контент».

内部自检，不要输出自检：
- Все 4 заголовка есть.
- Текст строго по теме {TOPIC}.
- Нет списков.
- Нет запрещенных фраз.
- Нет одинаковых абзацев.
- Нет финальной рекламной или мотивационной фразы.

现在生成 финальный русский текст.
```

```text
- Тема: {TOPIC}

Структура строго:
Введение
Теоретическая часть / Методы
Практическая часть / Эксперименты, результаты

Сделай текст похожим на работу аккуратного студента 2 курса:

不要解释你的方法。不要写计划。不要写 “готово”. Сразу выдай финальный русский текст.
```

## 5. Rewriting: универсальный промпт для фрагмента

Использовать, когда на сайте скачан исходный фрагмент. Лучше запускать в DeepSeek или Kimi.

```text
你要完成公开 CheckAI 比赛的「Переписывание фрагмента」任务。规则要求最终文本 полностью由 AI 生成。不要 просить меня редактировать текст вручную.

Итоговый язык: русский.
Исходная область: {DOMAIN}
Полная тема или название варианта: {TOPIC}
Требуемая секция, если известна: {SECTION}

Твоя задача:
1. Переписать исходный фрагмент как новую русскую научную секцию.
2. Сохранить смысл, тему и тип секции.
3. Не расширять это до полной статьи, если исходник является только одной секцией.
4. Объем итогового текста: не меньше 4000 символов, лучше 4500-6500.
5. Не использовать списки, таблицы, markdown, примечания, объяснения.
6. Не добавлять отдельный список литературы.
7. Если в исходнике есть заголовок секции, сохрани секционный тип, но формулируй текст заново.

Стиль:
- научный русский язык;
- текст студента 2 курса, а не профессора и не рекламного автора;
- больше конкретики по теме, меньше гладких универсальных фраз;
- абзацы разной длины;
- без искусственных ошибок;
- без длинных деепричастных цепочек;
- без однотипных начал предложений.

Нельзя использовать:
«В современном мире», «Следует отметить», «Важно понимать», «Исследования показывают», «Эксперты отмечают», «Таким образом, можно сделать вывод», «Подводя итог», «это позволяет значительно улучшить результаты».

Исходный фрагмент:
<<<
{FRAGMENT}
>>>

Перед ответом внутренне проверь:
- секция совпадает с исходной;
- тема сохранена;
- длина больше 4000 символов;
- нет списков;
- нет запрещенных фраз.

Выведи только финальный русский текст секции.
```

## 6. Повторная генерация после слабой оценки

Не использовать как ручное редактирование старого документа. Это запрос на новый независимый вариант.

```text
请不要 редактировать предыдущий текст. Сгенерируй полностью новый независимый вариант по той же теме с учетом диагностической информации.

Трек: {TRACK}
Область: {DOMAIN}
Тема: {TOPIC}
Диагностика прошлой отправки:
{SCORE_FEEDBACK}

Что изменить в новом варианте:
- Если была низкая релевантность, чаще используй точные термины из темы, но без механического повтора.
- Если была слабая структура, используй точные обязательные заголовки.
- Если была высокая ИИ-оценка, сделай менее симметричные абзацы, убери шаблонные переходы, добавь больше предметных деталей и естественных ограничений.
- Если было слабое качество, добавь метод, наблюдения, параметры, причинные связи и выводы.

Сохрани все базовые правила:
русский язык, 4000-100000 символов, научный стиль, без списков, без markdown, без запрещенных фраз, без ручной доработки после ответа.

Выведи только новый финальный русский текст.
```

## 7. Короткий тематический усилитель

Этот блок можно вставлять в любой промпт после темы.

```text
Предметная конкретика обязательна:
- Для инженерных и материаловедческих тем: материал, параметр процесса, дефект, причина, метод наблюдения, практическое ограничение.
- Для медицинских и биологических тем: механизм, показатель, группа пациентов/клеток/молекул, диагностическое ограничение, риск неверной интерпретации.
- Для права и социальных наук: норма/ситуация, конфликт интересов, критерий оценки, пример из практики без выдуманной ссылки.
- Для экономики и маркетинга: переменная, поведение агента, ограничение данных, простая модель или наблюдение.
- Для лингвистики и психологии: речевой материал, участники, показатель, ошибка измерения, интерпретация.
- Для географии, климата и астрономии: масштаб, объект, параметр наблюдения, источник неопределенности.
```
